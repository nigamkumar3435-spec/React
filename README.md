# React Learning & Projects Workspace

This repository contains a collection of React exercises, learning modules, components, and mini-projects.

## 📂 Project Structure

- **`17Bootstrap/`**: React integration with Bootstrap styling (`learning-bootstrap`).
- **`18 Fragments/`**: Practicing React Fragments, mapping, and rendering techniques (`learning-Fragments`).
- **`2 create-react-comp/`**: Basics of creating React components (`create-react-comp`).
- **`my-react-app/`**: Initial React app setup with Vite (`first-react-project`).
- **`Projects/`**: Mini-projects built with React:
  - **`Todo-app-version-one/`**: A Todo application built with React components and Bootstrap.
  - **`U-Bharat-clock-version-one/`**: A digital clock application displaying current time.

## 🚀 Getting Started

To run any of the projects locally:

1. Navigate into any project folder, for example:
   ```bash
   cd "Projects/Todo-app-version-one"
   ```

2. Install the required dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Library**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: CSS, Bootstrap
- **Language**: JavaScript (JSX)
